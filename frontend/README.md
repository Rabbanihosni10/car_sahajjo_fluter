## Getting Started

### Prerequisites
- Flutter SDK (^3.9.2)
- Dart SDK

### Installation

1. Install dependencies:
```bash
flutter pub get
```

2. Update API endpoint in `lib/config/app_config.dart`

3. Run the app:
```bash
flutter run
```
